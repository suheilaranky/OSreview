Lecture 7 High-Yield: Multimedia Operating Systems

  • Continuous media: audio/video must arrive on time.
  • Core conflict: timing guarantees vs limited resources.
  • Important algorithms: EDF, Rate Monotonic, LLF, SCAN-EDF, GSS.
  • Important formulas: S = M(B+1), Ui = ei/pi, Laxity, Tplay(s) ≥ (M+G)/rdt.

Lecture 8 Video: OS for Embedded Systems

Lecture 8 High-Yield: OS for Embedded Systems

  • Embedded system = specific function, limited resources, reactive behavior.
  • Key comparisons: microprocessor vs microcontroller, host vs target, Linux vs mClinux.
  • TinyOS focus: WSN, power saving, components, commands/events/tasks, FIFO scheduler.
  • Resource sharing in TinyOS: dedicated, virtualized, shared, arbiter.
